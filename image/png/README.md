## atch-zlib.png

Generated by
[pngattach](https://github.com/skeeto/scratch/tree/58470254f4a95cdf7a53888e405c851c21eb2cae/pngattach),
a program that allows you to embed attachments into a PNG file by inserting the
file name and contents in a private chunk of type `atCh`.

The attachment contained in this PNG file is [`message.txt`](./message.txt). In
the `atCh` chunk, it is zlib-compressed (the
[article](https://nullprogram.com/blog/2021/12/31/) calls it
"DEFLATE-compressed") with compression level 9 (`Z_BEST_COMPRESSION`), so it
starts with the zlib header `78 da`.

According to the author (see
https://nullprogram.com/blog/2021/12/31/#example-png):

> I produced it like so (with the help of
> [ImageMagick](https://www.imagemagick.org/)):
>
> ```console
> $ echo P3 1 1 1 0 1 0 | convert ppm:- resize 200 png:- | pngattach message.txt >atch-test.png
> ```
